+ 屏幕类型：
	+ 背光板
	+ 面板


	+ TN/VA/IPS----面板
		+ TN：响应时间快、对比度和颜色差——————————只玩FPS
		+ VA：可曲面，对比度高，颜色鲜艳但还原度差，响应时间慢（有拖影）
			+ HVA：快
		+ IPS：均衡，色彩强，响应一般
			+ manoIPS：颜色
			+ FastIPS
	+ LED/MiniLED-----背光板
		+ LED：主流
		+ LCD：


	+ OLED：更快、更高对比度、更薄，对纯黑可做做到，而LED不行，
		+ 即，LED是发光套磨具，OLED则是是否发光
		+ OLED色彩还原不足，寿命短
