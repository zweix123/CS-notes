## Usage

+ 论文不像书籍，某个话题下可能有很多篇，Zotero可以用来维护论文文件（操作系统的文件系统也行）
+ 内置多种文献文件类型的阅读器，All in One阅读，比如CAJ（这似乎也不是一定要用它的理由）
    + 通过插件实现划词翻译
+ 有浏览器插件，可以一键导入文献文件（这个是不是有点方便了）
+ 导出文献的BibTeX类型数据文件（这个就很有用了）
+ 其他各种插件

## Install
