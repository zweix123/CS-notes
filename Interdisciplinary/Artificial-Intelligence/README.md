机器学习Machine Learning是实现人工智能的手段，而深度学习只是机器学习中的一种特定方法。

+ 机器学习大致分为四类：
    + 监督学习Supervised Learning：
        + 输出、输出、标签
        + 解决分类（预测类别）和回归问题（预测连续实数值）
    + 无监督学习Unsupervised Learning：解决聚类问题
    + 半监督学习Semi-supervised Learning
    + 强化学习Reinforcement Learning
